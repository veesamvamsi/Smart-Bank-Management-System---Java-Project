# Smart-Bank-Management-System---Java-Project
# Overview
Smart Bank Management System is a console-based Java application designed to simulate real-world banking operations. It allows users to create customers, open different types of accounts, deposit and withdraw money, transfer funds, and view transaction history. The project is fully developed using Object-Oriented Programming principles.

# Objective

To apply Java OOP concepts in building a secure, structured, and interactive banking system with real banking workflows.

# Technologies Used
Programming Language: Java
Concepts: OOP (Encapsulation, Inheritance, Polymorphism, Abstraction)
Data Structures: ArrayList
User Interaction: Scanner class (Console UI)

# Key Features
Add new customers to the bank
Create Savings or Current accounts
Deposit & Withdraw funds with validation
Fund Transfer between accounts
Interest calculation for Savings Account
Maintain and display transaction history
Secure and structured data handling using OOP
